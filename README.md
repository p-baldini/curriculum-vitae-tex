# Paolo Baldini's Curriculum Vitae

Paolo Baldini's Curriculum Vitae with custom LaTeX style.

## Troubleshoot

- On MacOS, you may need to install the "classico" font. You can do it by running:
```
curl -O https://tug.org/fonts/getnonfreefonts/install-getnonfreefonts
sudo texlua install-getnonfreefonts
sudo getnonfreefonts classico
```

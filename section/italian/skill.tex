\section{Competenze trasversali}

\vspace{-\baselineskip}

\begin{multicols}{2}

\skill
    {Creatività e risoluzione di problemi}{-1}
    {
        Amo da sempre le sfide mentali e le continue prove affrontate durante il dottorato mi han permesso di affinare queste due capacità.
    }

\skill
    {Organizzazione}{-1}
    {
        Il lavoro ingegneristico richiede un elevato livello di organizzazione di dati e attività per essere svolto efficacemente.
    }

\skill
    {Crescita personale}{-1}
    {
        Cerco costantemente di imparare di più e di utilizzare le informazioni per produrre qualcosa di nuovo.
        Questa caratteristica mi ha spinto a scegliere il percorso dottorale.
    }

\skill
    {Lavoro di gruppo e ``leadership''}{-1}
    {
        L'impegno accademico richiede una collaborazione e una supervisione costanti.
        % Queste capacità sono risultate evidenti durante le varie investigazioni scientifiche e 
        Un chiaro risultato di questo aspetto sono le varie pubblicazioni e tesi di cui sono autore e/o supervisore.
    }

\skill
    {Autonomia e iniziativa}{-1}
    {
        Lavorare con molti gruppi diversi a volte richiede di prendere l'iniziativa e di operare in autonomia.
    }

\end{multicols}
